# recursive-replace

### Description

A program that allows a user to enter a string, a substring they wish to find and another string they wish to replace the found substring with

### Example

**Main String** - `Hi haters`
**Substring String** - `Hi`
**Replace String** - `Bye`

**Result** - `Bye Haters`

### How to run

- Clone repository
- Navigate into repository
- Run `python index.py`
